Returns the current time in milliseconds when LWIP\+\_\+\+TIMERS == 1 and NO\+\_\+\+SYS == 1. 

Returns the current time in milliseconds, may be the same as sys\+\_\+jiffies or at least based on it. Don\textquotesingle{}t care for wraparound, this is only used for time diffs. Not implementing this function means you cannot use some modules (e.\+g. TCP timestamps, internal timeouts for NO\+\_\+\+SYS==1).
