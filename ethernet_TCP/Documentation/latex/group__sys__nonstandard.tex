\doxysubsection{Detailed Description}
lw\+IP provides default implementations for non-\/standard functions. These can be mapped to OS functions to reduce code footprint if desired. All defines related to this section must not be placed in \mbox{\hyperlink{lwipopts_8h_source}{lwipopts.\+h}}, but in \mbox{\hyperlink{cc_8h_source}{arch/cc.\+h}}! These options cannot be \#defined in \mbox{\hyperlink{lwipopts_8h_source}{lwipopts.\+h}} since they are not options of lw\+IP itself, but options of the lw\+IP port to your system.
