The only thread function\+: Starts a new thread named \char`\"{}name\char`\"{} with priority \char`\"{}prio\char`\"{} that will begin its execution in the function \char`\"{}thread()\char`\"{}. The \char`\"{}arg\char`\"{} argument will be passed as an argument to the thread() function. The stack size to used for this thread is the \char`\"{}stacksize\char`\"{} parameter.
