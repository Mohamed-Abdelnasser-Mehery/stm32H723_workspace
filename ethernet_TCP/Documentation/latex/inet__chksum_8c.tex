\doxysubsection{Detailed Description}
Internet checksum functions.~\newline
 These are some reference implementations of the checksum algorithm, with the aim of being simple, correct and fully portable. Checksumming is the first thing you would want to optimize for your platform. If you create your own version, link it in and in your \mbox{\hyperlink{cc_8h_source}{cc.\+h}} put\+:

\#define LWIP\+\_\+\+CHKSUM your\+\_\+checksum\+\_\+routine

Or you can select from the implementations below by defining LWIP\+\_\+\+CHKSUM\+\_\+\+ALGORITHM to 1, 2 or 3.
