\doxysubsection{Detailed Description}
All defines related to this section must not be placed in \mbox{\hyperlink{lwipopts_8h_source}{lwipopts.\+h}}, but in \mbox{\hyperlink{cc_8h_source}{arch/cc.\+h}}! If the compiler does not provide memset() this file must include a definition of it, or include a file which defines it. These options cannot be \#defined in \mbox{\hyperlink{lwipopts_8h_source}{lwipopts.\+h}} since they are not options of lw\+IP itself, but options of the lw\+IP port to your system.
